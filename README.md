# d00r

Branded timelines for processes and projects. For anyone explaining what happens when.

Live at [d00r.app](https://d00r.app).

## What it is

A tool for building clean, shareable timelines. Put in your steps, get out a polished visual you can link to, screenshot, or embed. Useful for onboarding flows, project plans, service timelines, hiring processes — anywhere you'd otherwise reach for a slide deck.

## Why

Most timeline tools are either buried inside project management software or locked behind a login. d00r is meant to be the opposite: open a tab, build a timeline, share the link. No account.

## Status

Beta. The shape works, details are still moving.

## Stack

Single static HTML file. No backend. State is client-side only.
